## Installation 

```
$ git clone https://github.com/z3dc0ps/0x0p1n3r 
$ cd 0x0p1n3r
$ bash setup.sh
```

## Adding Virustotal API Key 
```

Go to https://www.virustotal.com/ and Login
Go to API key which in Top Right Option Menu
Copy API key and Paste in config.json

```

## Usage
```
$ bash run.sh
```
